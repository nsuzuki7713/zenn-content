## コマンド

```bash
# 新しい記事を作成する
$ npx zenn new:article

# 新しい本を作成する
$ npx zenn new:book

# 表示をプレビューする
$ npx zenn preview

# CLIをアップデートする
npm install zenn-cli@latest
```
